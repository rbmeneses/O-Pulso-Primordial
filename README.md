# O-Pulso-Primordial
A Teoria do Pulso Primordial não é uma resposta definitiva, mas uma ferramenta para pensar, um novo mapa para um território desconhecido. Ela propõe que as perguntas mais difíceis da cosmologia não podem ser respondidas dentro da nossa concepção mais intuitiva de realidade: a de um tempo linear que flui de um único começo para um único fim.
